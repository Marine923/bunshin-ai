🔒 あなたのデータは、この Mac から一歩も出ません

AI 推論もローカル (Ollama)。
Anthropic / OpenAI / Google など、外部サービスへの送信は一切ありません。 Anthropic API キー (任意) を設定すると describe 精度が上がりますが、 記録本体は Mac 内で完結します。

🔍

横断検索

Gmail / 写真 / メモ / Claude / ブラウザ履歴を一発で。意味検索 + キーワード検索のハイブリッド。日本語フル対応。

💬

あなたの記憶に基づくチャット

Ollama でローカル LLM が動作。過去の記録を文脈として自動で読み込み、引用元へ即ジャンプ。

🕸

関係性グラフ

あなたの世界を構成する人・プロジェクト・場所が蜘蛛の巣ビューで可視化。タイプ別に色分け。

📅

タイムライン

記録を日付別に整理。「先月の旅行で何があった？」を秒で振り返り。

💡

毎日の気づき

放置プロジェクト / 直近の予定 / 過去の質問が朝のフラッシュバックで届く。

🤖

MCP 対応

Claude Code / Claude Desktop から「あなたの記憶」を呼び出せる。7 個の MCP ツール公開。

※ bunshin.app との関係
同じ「Bunshin」名で bunshin.app という別プロダクトがあります。あちらは 開発者向けの Claude Code デスクトップラッパー。こちら Bunshin Memory は 一般ユーザー向けのパーソナル記憶 AI。カテゴリが完全に異なります。
